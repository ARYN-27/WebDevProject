C Variables, Constants and Literals

This course will teach you about variables and the rules for naming them. You'll also learn how to build constants and different types of literals in C programming.

Video: Variables in C Programming

Variables

A variable is a container (storage area) for data in programming.

Each variable should be given an unique name to denote the storage area (identifier). Variable names are nothing more than a symbol for a memory location. For example: int playerScore = 95;

Here, playerScore is a variable of int type. Here, the variable is assigned an integer value 95.

The term variable comes from the fact that a variable's value can be modified.

char ch = 'a';
// some code
ch = '1';

Rules for naming a variable

  1. A variable name can only have letters (both uppercase and lowercase letters), digits and underscore.
  2. The first letter of a variable should be either a letter or an underscore.
  3. There is no rule on how long a variable name (identifier) can be. However, you may run into problems in some compilers if the variable name is longer than 31 characters.

Note: You should always try to give meaningful names to variables. For example: firstName is a better variable name than fn.

C is a language with a lot of types. Which means that once a variable is declared, it cannot be modified. For example:

int number = 5; // integer variable
number = 5.5; // error
double number; // error

Here, the type of number variable is int. You cannot assign a floating-point (decimal) value 5.5 to this variable. Also, you cannot redefine the data type of the variable to double. By the way, to store the decimal values in C, you need to declare its type to either double or float.

Literals

Literals are data used for representing fixed values. They can be used directly in the code. For example: 1, 2.5, and 'c'.

Here, 1, 2.5 and 'c' are literals. Why? These terms cannot have various values assigned to them.

1. Integers

An integer is a numeric literal (a number) that does not contain any fractional or exponential parts. In C programming, there are three types of integer literals:

  • decimal (base 10)
  • octal (base 8)
  • hexadecimal (base 16)

For example:

        Decimal: 0, -9, 22 etc
        Octal: 021, 077, 033 etc
        Hexadecimal: 0x7f, 0x2a, 0x521

In C programming, octal starts with a 0, and hexadecimal starts with a 0x.

2. Floating-point Literals

A numeric literal with a fractional or exponent form is known as a floating-point literal. For example: 

        -2.0
        0.0000234
        -0.22E-5

Note: E-5 = 10-5

3. Characters

A character literal is created by enclosing a single character inside single quotation marks. For example: 'a', 'm', 'F', '2' and '}'.

4. Escape Sequences

In C programming, it is sometimes important to include characters that cannot be typed or have specific meaning. For instance, newline (enter), tab, and question mark (?).

Escape sequences are used to get these characters.

Escape Sequences
Escape Sequences Character
\b Backspace

\n Newline

\t Horizontal tab

\" Double quotation mark

\0 Null character

5. String Literals

A string literal is a collection of characters enclosed by double-quote marks. For example: 

"good"                    //string constant
""                       //null string constant
"      "                //string constant of six white space
"x"                    //string constant having a single character.
"Earth is round\n"    //prints string with a newline

Constants

The const keyword can be used to define a variable whose value cannot be modified. As a result, a constant will be created. For example: 


        const double PI = 3.14;

You'll notice that we've introduced the keyword const.

PI is a symbolic constant in this case, and its value cannot be modified. 

        const double PI = 3.14;
        PI = 2.9; //Error

Constant can also be defined using #define preprocessor directive.
